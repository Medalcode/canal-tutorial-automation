##SECCION: intro
¿Tienes un Android viejo en un cajón? No lo tires. Hoy lo convertimos en un servidor backend con Termux, Python y FastAPI.

##SECCION: instalacion
Primero, instala Termux desde F-Droid. No uses la versión de Play Store porque está desactualizada. Abre Termux y ejecuta estos comandos para actualizar e instalar lo necesario.

##SECCION: configuracion
Luego instala Python y OpenSSH. Con Python podrás correr tu backend y con SSH acceder de forma remota desde cualquier PC.

##SECCION: ssh
Ahora configura SSH. Ejecuta whoami para saber tu usuario, passwd para establecer una contraseña, e ifconfig para conocer la IP del teléfono en tu red local.

##SECCION: codigo
Conéctate desde tu PC con ssh usuario arroba ip. Ya conectado, crea un archivo server.py con FastAPI, importa FastAPI, crea la aplicación, define una ruta raíz que devuelva un mensaje, y ejecuta con uvicorn.

##SECCION: ejecucion
Ejecuta con uvicorn server punto app, host cero punto cero punto cero punto cero, puerto 8000. Abre tu navegador y visita http arroba ip del teléfono dos puntos 8000. Verás el mensaje en JSON.

##SECCION: cierre
Ya tienes un servidor funcionando 24 sietes que consume muy poca energía. En el próximo video haremos esto mucho más útil. Dale like y suscríbete para no perdértelo.
